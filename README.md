## Build Instructions

- Clone the [JUCE](https://github.com/juce-framework/JUCE "JUCE repository")
- Build _Projucer_
- Open the .jucer file in _Projucer_
- From here you can compile, run and debug the app from your native IDE

For more detailed instructions please refer to the [JUCE](https://github.com/juce-framework/JUCE "JUCE repository") and the 
[JUCE Tutorials](https://juce.com/learn/tutorials "JUCE tutorials").
